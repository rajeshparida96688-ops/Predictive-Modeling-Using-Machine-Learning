# 🤖 Predictive Modeling Using Machine Learning

## 📌 Overview

This project demonstrates how to build predictive machine learning models using the Titanic dataset. The objective is to preprocess data, train multiple supervised learning models, evaluate their performance, and visualize the results using confusion matrices and ROC curves.

---

## 🎯 Objectives

* Build predictive machine learning models
* Preprocess and prepare data for training
* Train and test classification algorithms
* Evaluate model performance using different metrics
* Visualize model performance

---

## ✨ Features

* ✅ Data Preprocessing
* ✅ Missing Value Handling
* ✅ Label Encoding

* ✅ Decision Tree Classifier
* ✅ Random Forest Classifier
* ✅ Accuracy Evaluation

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/rajeshparida96688-ops/Predictive-Modeling.git
```

Move to the project folder:

```bash
cd Predictive-Modeling
```

Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

---

## ▶️ Run the Project

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```
project.ipynb
```
